# Задача № 3. Выпуклые многоугольники.

Реализовать класс выпуклых многоугольников на плоскости:
1. Операции: пересечение, проверка на нахождение внутри, триангуляция;
2. Методы: площадь и периметр;
3. Проверка выпуклости при инициализации из списка вершин.

Исправления: 
1. Добавлена проверка на самопересекающиеся стороны с помощью метода _has_self_intersection
2. Добавлен метод _segments_intersect(): Проверяет пересекаются ли два отрезка и обрабатывает случаи коллинеарности
3. Добавлены самопересекающиеся и вонутые многоугольники в качестве тесотвых данных

# Запуск проекта:

Скачайте к себе на компьютер файл проекта

```
git clone https://github.com/Polytech-Python-Team/task-3-convex-polygons.git
```

- Установите язык программирования [Python](https://www.python.org/downloads/)
- Откройте консоль в папке проекта и создайте там виртуальное окружение

```
python -m venv venv
```

- Активируйте данное виртуальное окружение
- Для Windows:

```
venv\Scripts\activate
```

- Для Linux:

```
source venv/bin/activate
```

- Установите все необходимые модули для работы приложения
